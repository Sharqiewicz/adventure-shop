### Installation

1. Install virtualenv

    - asgiref==3.2.7
    - Django==3.0.7
    - Pillow==7.2.0
    - pytz==2020.1
    - sqlparse==0.3.1
2. console: source bin/activate
3. console: python3 manage.py runserver
